##Rally Plugin

This Plugin will install Rally and run the Live Migration Plugin. 

#**Steps**

1. Clone the repo in /opt/ownPlugin folder.
'git clone https://github.com/enamshah09/RallyPlugin.git /opt/ownPlugin'
2. cd /opt/ownPlugin
3. 'mv setupLvmEnv.sh /opt'
4. 'mv vars.rc /opt'
5. cd /opt
6. './setupLvmEnv.sh'
